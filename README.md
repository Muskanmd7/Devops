

## Features
- Flask Web App
- Dockerized Application
- Deployed on AWS EC2
- CI/CD using Jenkins
- Configuration using Ansible

## Run Locally
docker build -t devops-app .
docker run -p 5000:5000 devops-app

## Tech Stack
- Python (Flask)
- Docker
- AWS EC2
- Jenkins
- Ansible
